List of commands
  • help Shows available commands and usage notes.
  • add Adds a patient with prefixed fields.
  • list Displays all patients, or filters them by urgency.
  • find Searches for patients by name in the current records.
  • update Updates the selected patient's details.
  • delete Removes a patient from the current list.
  • undo Restores the last data-changing action.
  • clear Clears the current filtered view.
