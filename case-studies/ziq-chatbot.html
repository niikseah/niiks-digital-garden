Ziq

An in-built simulator of Ziq, a task chatbot.

Tool
Java

List of Commands

  • help - show all available commands
  • list - show current tasks
  • todo <task> - add a todo task
  • deadline <task> /by <DDMMYYYY> [<HHmm>] - add a deadline task
  • event <task> /from <DDMMYYYY> <HHmm> /to <DDMMYYYY> <HHmm> - add an event task
  • mark <index> / unmark <index> - update task status
  • delete <index> - remove a task
  • find <keyword> - search tasks by keyword
  • tag <index> <tag> - add a tag to task
  • organise tag / organise deadline - organize task list
  • schedule <DD|DDMM|DDMMYYYY> - show tasks for a date
  • clear - remove all tasks
  • reset - reset the current session
